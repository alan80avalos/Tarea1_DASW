AARA

GitHub es una plataforma en la nube, muy utilizada por los desarrolladores, ya que sirve para alojar proyectos utilizando un sistema de control de versiones en la nube, de forma que evita pérdida de datos y facilita la accesibilidad. Además ayuda para el seguimiento y gestión de los cambios en el código de manera eficiente.

Comandos principales:

Comando para iniciar un repositorio en Git:
-Git init
Comando para clonar un repositorio:
-git clone + (dirección del repositorio)
Comando para ver el estado del repositorio:
-git status
Comando para añadir cambios:
-git add + (nombre del archivo)
Comando para realizar el comit:
-git commit -m + (La descripción de lo realizado) <- entre comillas
Comando para ver el historial de cambios:
-git log
Comando para subir los cambios al repositorio:
-git push origin main
Comando para obtener los cambios:
-git pull origin main
Comando para crear una nueva rama:
-git branch + (nombre de la rama)
Comando para cambiar de rama:
-git checkout + (nombre de la rama)
Comando para eliminar una rama:
-git branch -d + (nombre de la rama)
